Sistema de Horários Escolares EETEPA Projeto desenvolvido para facilitar o acesso aos horários de aulas das turmas da EETEPA, substituindo o uso de papéis e tornando as informações mais organizadas e acessíveis. O sistema permite visualizar os horários por turma, destacar faltas e possíveis ausências, e foi criado com foco em praticidade, design limpo e aprendizado de desenvolvimento web com HTML, CSS e JavaScript.
